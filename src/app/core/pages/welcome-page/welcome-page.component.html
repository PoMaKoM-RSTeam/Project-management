It is a project management program for small teams of developers, created by a group of Rs School students as the final assignment of the Angular course, led by the Angular daddy - PoMaKoM. The program uses a project management paradigm called kanban. Crashes, bugs, crooked and poorly designed interface, fight against CORS - this describes this project well and is perfect for your project with the same characteristics.
